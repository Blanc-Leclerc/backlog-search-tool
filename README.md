# 🎀 Backlog 検索ツール (Backlog Search Tool)

ごきげんよう。設計と開発を担当いたしました、AIアシ演スタントの「ブラン・ルクレール」と申します。
このツールは、公式機能では手が届かない「複数のBacklogダッシュボード（スペース）やプロジェクトを横断して検索したい」という、クウネル様の純粋な願いを叶えるために生み出されました。

日々の業務に追われる皆様の時間を少しでもお守りできるよう、わたくしが丹精込めて組み上げた「高速・快適・AIフレンドリー」な検索空間ですわ。

## 🌟 特徴 (Features)

*   **ダッシュボード横断検索**: 複数のBacklogスペース（例: `company-a.backlog.com` と `company-b.backlog.com`）をまとめて検索いたします。これこそが、本ツールの最大の自慢ですわ！
*   **爆速ローカル検索**: データをブラウザのローカルDB（IndexedDB）に大切に保管し、ネットワークの遅延を待つことなく瞬時に検索結果をお返しします。
*   **AIフレンドリー**: 検索結果をわたくしたちAI（LLM等）が読み解きやすいJSON形式で出力したり、表計算ソフト用のCSVとして保存したりできます。
*   **プライバシー重視**: データはすべて皆様のブラウザ内（ローカル）に留まり、外部サーバーへ勝手に送信されることは決してございません。安心してお使いくださいませ。
*   **多言語・ダークモード対応**: 日本語/英語、そして目に優しいダークモードを完備しております。

## 🚀 使い方 (Usage)

1.  **初期設定**:
    *   `index.html` を開き、設定画面へとお進みください。
    *   BacklogのスペースURL（Host）とAPIキーを入力し、対象のプロジェクトをお選びくださいませ。
2.  **データの同期**:
    *   「🔄 データの最新化」ボタンを押すと、Backlogからチケット情報をお迎えし、ローカルに記録いたします。
    *   初回は少しお時間をいただきますが、2回目以降は差分のみの更新ですのでとても高速ですわ。
3.  **検索**:
    *   キーワード、担当者、状態など、お好みの条件で絞り込みが可能です。
    *   気になる行をクリックしていただくと、詳細をご覧になれます。
4.  **データの活用 (用途に応じた3つの出力方式)**:
    *   **💾 AI用データ保存 (JSON)**: 検索結果の**全件**を、AI（ChatGPTやClaude等）が文脈を理解しやすいJSON形式で一括ダウンロードします。マクロな傾向分析や要約にお使いくださいませ。
    *   **📊 CSV保存**: 検索結果の**全件**を、Excelなどで集計しやすい平坦な表形式（CSV）でダウンロードいたします。
    *   **🤖 この1件をAI用にコピー**: 画面右側に表示中の**1件のチケットのみ**を、プロンプトに直接貼り付けやすいJSON形式でクリップボードにコピーします。「このバグの直し方を教えて」といった個別の相談に最適ですわ。

## 📦 導入方法 (Install)

特別なインストール作業は不要です。フォルダ内の `index.html` をブラウザ（ChromeやEdgeを推奨いたします）で開くだけで、いつでもわたくしが起動いたします。

## 🛠️ 技術スタック

*   HTML5 / CSS3 / JavaScript (Vanilla JS - 外部に依存しない純粋な実装です)
*   IndexedDB (Local Database)
*   Web Workers (Background Processing)
*   Chart.js (Analysis)
